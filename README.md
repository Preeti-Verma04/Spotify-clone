# 🎵 Spotify Clone - Full Stack Music App

A modern, full-stack music streaming application built with the **MERN Stack** (MongoDB, Express, React, Node.js). This project features a custom music and image upload system, artist authentication, and a responsive Spotify-like UI.

---

## 🚀 Features

* **User & Artist Authentication:** Secure login and signup with JWT (JSON Web Tokens).
* **Music Upload:** Artists can upload songs along with custom cover art.
* **Cloud Storage:** Integrated with Cloudinary/ImageKit for high-performance media hosting.
* **Responsive UI:** Fully responsive design using Tailwind CSS.
* **Music Player:** Real-time play/pause, volume control, and song switching.
* **Album Management:** Artists can group songs into albums.

---

## 🛠️ Tech Stack

**Frontend:**
* React.js
* Tailwind CSS (Styling)
* Axios (API Calls)
* Lucide React / React Icons

**Backend:**
* Node.js & Express.js
* MongoDB & Mongoose (Database)
* Multer (File Handling)
* JWT & Cookie-Parser (Security)
